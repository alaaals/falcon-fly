Welcome to My Game!

Directions:
Use the 'A' and 'D' keys to move left and right, and collect as much coins as possible while evading the hurtful objects!

Description of Game:
Eagle Dodge is a game where the user plays as an eagle flying through the sky and collecting as many coins as possible. Each coin collections increases a player’s score by 1. There will be, at randomly chosen locations, flying obstacles that the player would have to avoid while collecting these coins, such as knifes, rocks, and fireballs. Each hit from those 'hurtful' looking objects will result in the health-bar of the player decreasing by one. In total, a user has 7 lives before the game is over. There are three modes of difficulty in this game, 'Easy', 'Medium', and 'Hard'. Switching between these modes will make the speeds of both the coins and the obstacles start to increase more, making it more difficult for the player to dodge obstacles and/or collect coins. There are three states/screens to this game, a 'starter' screen, where the player will be able to hit the start button in order to get the game started, a 'playing' screen, where the actual game will take place, and finally, an 'end' screen, where the player will be transported to once the game is over (all their lives are used up). Additionally, high-scores will be recorded, stored, and displayed at both the starter screen and the end screen each time a player beats the old score.

This game was inspired by relatively simple mobile games such as Flappy Bird and other endless flying games. What’s a bit different about this particular game is that, unlike the ones where the player is able to play almost infinitely upwards (except if they fall down or miss a step), this game gives a player a limited amount of lives that decreases each time it hits the wrong object. This, plus the randomness of where the objects start falling, gives the game an added challenge in order to play for longer periods of time.
